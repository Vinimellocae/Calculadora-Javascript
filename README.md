# 🧮 Calculadora Web

Calculadora interativa desenvolvida com **JavaScript puro (Vanilla JS)**, capaz de realizar operações matemáticas básicas com suporte a números decimais e tratamento de erros.

---

## 📌 Sobre o projeto

Este projeto simula uma **calculadora funcional**, permitindo que o usuário realize operações como soma, subtração, multiplicação e divisão diretamente no navegador.

A aplicação foi construída com foco em:
- Manipulação do DOM
- Controle de estado da aplicação
- Tratamento de entradas do usuário

---

## 🚀 Funcionalidades

- ➕ Soma
- ➖ Subtração
- ✖️ Multiplicação
- ➗ Divisão
- 🔢 Suporte a números decimais (vírgula)
- ⬅️ Botão de apagar (backspace)
- 🧹 Limpar tudo (AC)
- ⚠️ Tratamento de erro (divisão por zero)
- 🔄 Atualização dinâmica do display

---

## 🛠️ Tecnologias utilizadas

- HTML
- CSS
- JavaScript (Vanilla JS)
